# Fluon

> AI-powered productivity companion built for the **Vibe2Ship Hackathon 2026**.

Fluon helps students, professionals, and entrepreneurs stay ahead of deadlines through intelligent planning, AI-powered insights, adaptive scheduling, and emotional support.

---

# Problem Statement

**Problem Statement 1 – The Last-Minute Life Saver**

Traditional productivity applications rely on passive reminders that are easy to ignore.

Fluon goes beyond reminders by proactively helping users plan, prioritize, and complete tasks before deadlines are missed using Google Gemini.

# Installation

Clone the repository

```bash
git clone https://github.com/chandanadeyyala-crypto/Fluon.git
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

# Why Fluon?

Most productivity tools remind users after deadlines become urgent.

Fluon actively assists users by:

- understanding context
- generating plans
- prioritizing work
- reducing burnout
- providing emotional support
- adapting recommendations over time

The objective is not simply task management—but helping users consistently complete meaningful work while maintaining a healthy balance.
